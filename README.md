# Remyn

A clean and lightweight reminder app for Android and iOS. Set reminders your way — by time interval, specific date, or weekday — and stay on top of what matters.

## Features

- **Quick Reminders** — Create reminders with a title and optional description in seconds.
- **Flexible Scheduling** — Choose from time intervals, specific date and time, or recurring weekdays.
- **Repeating Reminders** — Set reminders to repeat so you never miss a recurring task.
- **Personalization** — Assign custom icons and colors to organize and distinguish your reminders at a glance.
- **Dark Mode** — Switch between light and dark themes to match your preference.
- **Offline Storage** — All data is stored locally on your device. No account or internet required.

## Contributing

Want to contribute? You are welcome to. Follow these steps to get started:

1. Fork this repository.
2. Clone your fork locally.
   ```bash
   git clone https://github.com/<your-username>/remyn.git
   ```
3. Install dependencies.
   ```bash
   flutter pub get
   ```
4. Create a new branch for your changes.
   ```bash
   git checkout -b your-branch-name
   ```
5. Make your changes and test them.
6. Commit and push to your fork.
   ```bash
   git add .
   git commit -m "Brief description of your changes"
   git push origin your-branch-name
   ```
7. Open a Pull Request against the `main` branch of this repository.
